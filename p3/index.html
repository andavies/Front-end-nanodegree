Frogger game


Object-oriented Javascript



This game was the final project for a course in object-oriented Javascript, delivered by Hackreactor's Marcus Phillips and Udacity.


The player and enemy objects were implemented first in the functional style, then prototypal, and finally as pseudoclassical classes. You can see the commit history and source code here.


It's pretty basic at the moment and has a few minor bugs (no), but I plan to come back to it and add features such as collectables and points, but for now it demonstrates my use of Javascript and HTML5 Canvas.


Note that I've disabled scrolling with the arrow keys so that it doesn't interfere with the game, so use your mouse!
